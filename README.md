# Wiki de Mods de Minecraft 🎮

Proyecto final hecho para la materia de **Fundamentos de Tecnologías Web (2026)** en la Universidad Veracruzana.

Inspirado en la interfaz y experiencia de usuario de **CurseForge**, este sitio web es una wiki responsiva estructurada en modo oscuro que detalla dinámicamente las mecánicas, biomas y secretos de los mods más populares de Minecraft.

* **Sitio Web (GitHub Pages):** [Ingresa aquí a la Wiki](https://antoniotejedalp-jpg.github.io/ordinario-ftw//)
* **Video Demostrativo (YouTube):** [Ver navegación y explicación](https://youtu.be/no9Nu283l_U) 

## 🛠️ Tecnologías Utilizadas

* **HTML5 Semántico:** Uso estricto de regiones semánticas (`<nav>`, `<main>`, `<article>`, `<section>`) y optimización de accesibilidad (atributos `alt` y etiquetas asociadas).
* **CSS3 Personalizado:** Diseño responsive basado en variables globales, rejillas adaptativas y *Media Queries* para soporte móvil.
* **JavaScript Asíncrono (ES6):** Manipulación del DOM mediante plantillas literales y consumo de datos mediante la API `fetch`.
* **XML + DTD:** Base de datos local validada mediante un esquema DTD interno para la persistencia del catálogo de mods y control de acceso.

## 📋 Características Principales

1. **Buscador Dinámico:** Filtrado en tiempo real de mods por nombre o categoría sin recargar la página.
2. **Control de Acceso:** Sistema de Login con validación cruzada de credenciales contra los nodos registrados en el archivo XML.
3. **Navegación Accesible:** Interfaz 100% operable mediante teclado (uso secuencial de Tabulador) y menús adaptativos para celulares.
